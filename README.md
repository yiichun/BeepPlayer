🎹 BeepPlayer - 互動式鋼琴模擬器

📌 專案簡介

本專案為一個基於 C# Windows Forms 開發的互動式鋼琴程式。透過呼叫 Windows 底層系統 API (kernel32.dll 中的 Beep)，實作了精準的音頻發聲功能。

🛠 執行說明

1. 使用滑鼠點擊畫面上的白色按鈕（自然音）或黑色按鈕（半音）進行彈奏。

2. 按鈕按下時會有視覺顏色回饋，發聲完畢後會自動恢復原色。

3. 可任意拖動視窗邊緣改變大小，介面將自動適應。
